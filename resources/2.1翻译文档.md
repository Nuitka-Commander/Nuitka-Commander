### --noinclude-custom-mode=CUSTOM_CHOICES

- [ ] 选项完成
- [ ] 链接完成

```
--noinclude-custom-mode=CUSTOM_CHOICES
```

```
不包含自定义模式=自定义选项
```

原始简介:

```
What to do if a specific import is encountered. Format is module name, which

```

```

例如PyQt5:error。
```
